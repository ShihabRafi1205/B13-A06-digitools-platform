Project Name : Digitools Platform.

Description : It's a Frontend E-commarce UI , Product Listing and Cart-System that offers to clients access to premeum Digital Tools for their smooth workflow and productivity.

Technologys : Used here basically 
React.js
Tailwind CSS, DaisyUI
JavaScript (ES6+)
React-Toastify (NPM Package)
and others React-packages

Features : 
 1. One of the beatiful features in this project is : Toggle functionality that means By Clicking Product, the Products section has shown and By clicking Cart, the cart section has shown. By default it's just an empty message , By default Production visible all time.
 2.Buy Now Button : By Clicking Buy Now Button It add this card in cart section and the Remove Button remove the card.
 3. The beauty of this project is "Proceed to Checkout" it Clears all cart items by making array empty.
